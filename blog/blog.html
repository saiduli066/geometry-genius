1.What is DOM and what is the purpose of DOM?

Answer:DOM means Document Object Model. DOM allows to create, change, or remove elements from a website document.

2.How will you select HTML elements using DOM? Name the DOM methods.

Answer:There are some methods to select HTML elements using DOM.The methods
1.getElementsByTagName()
2.getElementsByClassName()
3.getElementById()
4.querySelector()
5.querySelectorAll()

3.What is event bubble?

Answer:Event bubble is a method of event propagation in the HTML DOM API where an event is intercepted by an element inside another element and handled by both elements.

4.What is a callback function and why will you use it?

Answer:Callback functions are passed into another function as arguments, which then execute some kind of action or function inside the outer function.
